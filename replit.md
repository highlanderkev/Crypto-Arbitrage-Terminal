# ArbitrageX Terminal

A crypto cross-exchange arbitrage AI research agent. Monitors price spreads across 7 major exchanges, surfaces live arbitrage opportunities, and uses an AI agent (GPT-5.4) to generate detailed research reports on each opportunity.

## Run & Operate

- `pnpm --filter @workspace/api-server run dev` — run the API server (port 8080)
- `pnpm --filter @workspace/arb-agent run dev` — run the frontend (port 21355)
- `pnpm run typecheck` — full typecheck across all packages
- `pnpm run build` — typecheck + build all packages
- `pnpm --filter @workspace/api-spec run codegen` — regenerate API hooks and Zod schemas from the OpenAPI spec
- `pnpm --filter @workspace/db run push` — push DB schema changes (dev only)
- Required env: `DATABASE_URL`, `AI_INTEGRATIONS_OPENAI_BASE_URL`, `AI_INTEGRATIONS_OPENAI_API_KEY`

## Stack

- pnpm workspaces, Node.js 24, TypeScript 5.9
- API: Express 5
- Frontend: React + Vite + Tailwind + shadcn/ui + wouter
- DB: PostgreSQL + Drizzle ORM
- Validation: Zod (`zod/v4`), `drizzle-zod`
- AI: OpenAI GPT-5.4 via Replit AI Integrations (no user key needed)
- API codegen: Orval (from OpenAPI spec)
- Build: esbuild (CJS bundle)

## Where things live

- `lib/api-spec/openapi.yaml` — API contract (source of truth)
- `lib/db/src/schema/` — Drizzle schema for all tables
- `artifacts/api-server/src/routes/` — Express route handlers
- `artifacts/api-server/src/lib/arbitrageEngine.ts` — simulated price & opportunity scanner
- `artifacts/api-server/src/lib/seed.ts` — DB seeding (exchanges, watchlist, sample report)
- `artifacts/arb-agent/src/` — React frontend (pages, components)

## Architecture decisions

- **Simulated exchange prices**: Real exchange APIs require API keys and have rate limits. The engine simulates realistic price data with per-exchange biases, making the arb detection real and demonstrable without external dependencies.
- **Opportunity TTL**: Active opportunities expire after 5 minutes; a background scan every 2 minutes generates fresh opportunities to simulate a live feed.
- **AI research via OpenAI**: Research reports are generated by GPT-5.4 with structured JSON output, then stored in DB for fast retrieval.
- **SSE streaming chat**: The `/chat` page uses raw fetch + ReadableStream (not generated hooks) since Orval can't type SSE streaming endpoints.
- **Contract-first**: OpenAPI spec gates all codegen → no hand-written types that can drift.

## Product

- **Dashboard**: Live summary stats, top 10 opportunities, top symbols and exchange pairs
- **Opportunities**: Full searchable/filterable live opportunity table with spread color coding
- **Opportunity Detail**: Price breakdown, fee analysis, net profit estimate, trigger AI research
- **Research Reports**: Browse AI-generated analyses with confidence scores and recommendations
- **Price Monitor**: Real-time price grid across all exchanges, filterable by symbol
- **Watchlist**: Track specific crypto pairs
- **AI Chat**: Conversational arbitrage research assistant (streaming)

## Gotchas

- Always run codegen after OpenAPI spec changes: `pnpm --filter @workspace/api-spec run codegen`
- The arbitrage scan runs on startup and every 2 minutes; first load may show 0 opportunities for ~1 second
- `react-markdown` and `remark-gfm` are installed in `arb-agent` for markdown rendering in research reports
